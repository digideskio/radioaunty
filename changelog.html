RadioAunty 2.1.1

  • Fix Schedule Bug

    Some local schedules were not displaying correctly due to a bug in the formatting. This has now been fixed.

RadioAunty 2.1

  • Listen to all BBC Stations

    You can now listen to ANY BBC Radio station, including both BBC National and Local radio. You can also customise which of these stations appear in the Listen Menu. This is done via the updated Preferences window.

  • Full 7 Day catch

    You can now simply navigate through the FULL 7 day catchup window for each station via the Schedules menu. If the programme is clickable, then you can listen to it. If you hover over any programme you will see extra information about the show, including how long there is left to listen.
